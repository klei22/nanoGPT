What you are seeing

Each yellow vector starts on the unit sphere. In the next stage, each one is multiplied by its own scalar in the range [0, 2].

The blue chain shows tip-to-tail addition, which forms the raw summed vector. A faint blue arrow from the origin shows the same total displacement.

In the last stage, the raw sum is projected back to radius 2, leaving only its direction as the normalized output vector.
