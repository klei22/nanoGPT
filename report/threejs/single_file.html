Read → Activate → Write, then Quantize → Read → Activate → Write

Two separate vector banks are initialized: a read bank and a write bank. The read bank produces coefficients from dot products with the incoming vector. Those coefficients are transferred to the write bank by shared index, then used to scale, sum, and normalize. A second pass rotates both banks to the nearest quantized sphere directions and repeats the same pipeline.
