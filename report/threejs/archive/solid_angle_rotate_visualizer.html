Read → Activate → Write, then Random-Rotate Solid Angles → Read → Activate → Write

Two separate vector banks are initialized: a read bank and a write bank. The read bank produces coefficients from dot products with the incoming vector. Those coefficients are transferred to the write bank by shared index, then used to scale, sum, and normalize. In the second pass, both banks rotate toward fixed random sphere targets while every vector is shown as a transparent solid-angle cone with an arrow at its center. If cones overlap while moving, the system applies tangential separation so they slide around each other instead of bouncing.
