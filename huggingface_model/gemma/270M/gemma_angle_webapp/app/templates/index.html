Minimum angular distance

Closest non-self token for every token

For each token vector, this streams blockwise dot products, excludes the token’s self-distance, keeps only the closest other token, and never writes or caches the full pairwise matrix.

Load a model, then compute each token’s closest non-self angular neighbor. CUDA is used automatically when available.

Y axis is the minimum non-self angle. X axis is rank after sorting tokens from lowest to highest minimum angle.

Recursive angle group

Grow a connected token group by maximum angle

Starting from a seed token, this repeatedly adds tokens whose signed vector angle is at or below the chosen maximum from any token already in the group. It stops when no new tokens are found or the group reaches the size limit.
